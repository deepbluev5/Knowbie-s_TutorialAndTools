重新介绍J ava Script（JS教程）

为什么要重新介绍？因为JavaScript是世界上最容易被误解的编程语言而臭名昭着  。它常常被人嘲笑为玩具，但在其简单的欺骗性层面之下，有着强大的语言特色。JavaScript现在被众多备受瞩目的应用程序所使用，这表明对于任何Web或移动开发人员来说，深入了解这项技术是一项重要技能。

首先概述语言的历史，这很有用。JavaScript是由Brendan Eich于1995年在Netscape担任工程师时创建的。JavaScript早在1996年初就与Netscape 2一起发布了。它最初将被称为LiveScript，但它在一个命运多marketing的营销决策中被重新命名，试图利用Sun Microsystem的Java语言的流行 - 尽管它们有很少共同。自那以后，这一直是一个混乱的来源。

几个月后，微软发布了带有Internet Explorer 3的JScript。这是一个大体上兼容的JavaScript。几个月后，Netscape将JavaScript提交给了欧洲标准化组织Ecma International，该标准组织于当年推出了第一版ECMAScript标准。该标准在1999年作为ECMAScript第3版获得了重大更新，从那以后它一直保持稳定。由于语言复杂性的政治差异，第四版被放弃。第四版的许多部分构成了2009年12月出版的ECMAScript第5版和2015年6月出版的第6版主要版本的基础。

因为它比较熟悉，所以从这一点开始我们将ECMAScript称为“JavaScript”。

与大多数编程语言不同，JavaScript语言没有输入或输出的概念。它被设计为在主机环境中作为脚本语言运行，并且主机环境提供与外部世界进行通信的机制。最常见的主机环境是浏览器，但JavaScript解释器也可以在许多其他地方找到，包括Adobe Acrobat，Adobe Photoshop，SVG图像，Yahoo的Widget引擎，服务器端环境，如Node.js，NoSQL数据库如开源的Apache CouchDB，嵌入式计算机，完整的桌面环境，如GNOME（GNU / Linux操作系统最流行的GUI之一）等等。

概观

JavaScript是一种多范型动态语言，包含类型和操作符，标准的内置对象和方法。它的语法基于Java和C语言 - 这些语言的许多结构也适用于JavaScript。JavaScript支持对象原型而不是类的面向对象编程（请参阅有关原型继承和ES2015 类的更多信息）。JavaScript也支持函数式编程 - 函数是对象，给函数提供了保存可执行代码的能力，并像其他任何对象一样被传递。

我们先看看任何语言的构建块：类型。JavaScript程序操作值，这些值都属于一个类型。JavaScript的类型是：

  • Symbol （ES2015新增）

数字

根据规范，JavaScript中的数字是“双精度64位格式IEEE 754值”。这有一些有趣的结果。在JavaScript中没有这样的整数，所以如果你习惯用C或Java来进行数学运算，那么你必须小心一点。

另外，请注意以下内容：

0.1 + 0.2 == 0.30000000000000004;

实际上，整数值被视为32位整数，有些实现甚至以这种方式存储它，直到它们被要求执行一个在数字上有效但不在32位整数上的指令。这对于按位操作很重要。

支持标准算术运算符，包括加法，减法，模数（或余数）算术等等。还有一个内置对象，我们之前忘记提到Math它提供了高级数学函数和常量：

Math.sin(3.5);
var circumference = 2 * Math.PI * r;

您可以使用内置parseInt()函数将字符串转换为整数。这将转换的基础作为可选的第二个参数，您应该始终提供：

parseInt('123', 10); // 123
parseInt('010', 10); // 10

在较老的浏览器中，以“0”开头的字符串被假定为八进制（基数8），但自2013年以来这种情况并非如此。除非您确定了字符串格式，否则您可以在这些较旧的浏览器上获得令人惊讶的结果：

parseInt('010');  //  8
parseInt('0x10'); // 16

在这里，我们看到该parseInt()函数将第一个字符串视为前导0，并将第二个字符串视为十六进制，因为前导“0x”。在十六进制表示仍不到位 ; 只有八进制已被删除。

如果您想将二进制数转换为整数，只需更改基数：

parseInt('11', 2); // 3

同样，您可以使用内置parseFloat()函数来分析浮点数。与其parseInt()堂兄弟不同，parseFloat()总是使用10。

您也可以使用一元运算+符将值转换为数字：

+ '42';   // 42
+ '010';  // 10
+ '0x10'; // 16

NaN如果字符串是非数值型的，则会返回一个特殊值（简称为“不是数字”）：

parseInt('hello', 10); // NaN

NaN是有毒的：如果您将它作为操作数提供给任何数学运算，结果也将是NaN：

NaN + 5; // NaN

您可以测试NaN使用内置isNaN()函数：

isNaN(NaN); // true

JavaScript也有特殊的价值Infinity和-Infinity：

1 / 0; //  Infinity
-1 / 0; // -Infinity

您可以测试Infinity，-Infinity和NaN值使用内置isFinite()功能：

isFinite(1 / 0); // false
isFinite(-Infinity); // false
isFinite(NaN); // false
该parseInt()和parseFloat()功能解析字符串，直到它们到达一个字符是无效的指定数字格式，然后返回解析到这一点的数量。但是，NaN如果在其中包含无效字符，“+”运算符会简单地将字符串转换为字符串。只要在控制台中用自己的方法尝试解析字符串“10.2abc”，你就会更好地理解差异。

字符串

JavaScript中的字符串是Unicode字符的序列。对于那些不得不应对国际化的人来说，这应该是一个值得欢迎的消息。更准确地说，它们是UTF-16代码单元的序列; 每个代码单元由一个16位数字表示。每个Unicode字符由1或2个代码单元表示。

如果要表示单个字符，则只需使用由该单个字符组成的字符串。

要查找字符串的长度（以代码为单位），请访问其length属性：

'hello'.length; // 5

我们的第一个JavaScript对象刷！我们有没有提到你也可以使用类似对象的字符串？它们也有一些方法可以让你操纵字符串并访问关于字符串的信息：

'hello'.charAt(0); // "h"
'hello, world'.replace('hello', 'goodbye'); // "goodbye, world"
'hello'.toUpperCase(); // "HELLO"

其他类型

JavaScript之间的区别在于null，它是一个表示故意的非值（并且只能通过null关键字访问）undefined的值，并且它是undefined指示未初始化值的类型值 - 即，还没有分配值然而。稍后我们将讨论变量，但是在JavaScript中可以声明一个变量而不用为其赋值。如果你这样做，变量的类型是undefined。undefined实际上是一个常数。

JavaScript有一个布尔类型，可能的值true和false（都是关键字）。根据以下规则，可以将任何值转换为布尔值：

  1. false，0空字符串（""）NaN，null和undefined所有成为false.
  2. 所有其他值变成 true.

您可以使用以下Boolean()函数明确执行此转换：

Boolean('');  // false
Boolean(234); // true

然而，这是很少需要的，因为JavaScript会在默认情况下执行此转换，例如在if语句中（如下所示）。出于这个原因，我们有时简单地说出“真实值”和“虚假值”，这些值分别表示转换为布尔值时的值true和false值。或者，这些值可分别被称为“真理”和“虚假”。

布尔操作，如&&（逻辑和），||（逻辑或）和!（逻辑非）支持; 见下文。

变量

在JavaScript中新变量的声明使用的三个关键词之一：let，const或var。

let 允许你声明块级变量。声明的变量可以从它所在的块中获得。

let a;
let name = 'Simon';

以下是使用以下变量声明的变量的作用域示例let：

// myLetVariable is *not* visible out here

for (let myLetVariable = 0; myLetVariable < 5; myLetVariable++) {
  // myLetVariable is only visible in here
}

// myLetVariable is *not* visible out here

const允许你声明其值永远不会改变的变量。该变量可从其声明的块中获得。

const Pi = 3.14; // variable Pi is set 
Pi = 1; // will throw an error because you cannot change a constant variable.


var是最常用的声明性关键字。它没有其他两个关键字所具有的限制。这是因为它传统上是在JavaScript中声明变量的唯一方法。使用var 关键字声明的变量可以从其声明的函数中获得。

var a; 
var name = 'Simon';

变量声明为范围的示例  var：

// myVarVariable *is* visible out here 

for (var myVarVariable = 0; myVarVariable < 5; myVarVariable++) { 
  // myVarVariable is visible to the whole function 
} 

// myVarVariable *is* visible out here

如果你声明一个变量而不给它赋值，它的类型就是undefined。

JavaScript和其他语言（如Java）之间的一个重要区别是在JavaScript中，块没有范围; 只有函数有一个范围。所以如果一个变量是var在一个复合语句中定义的（例如在一个if控制结构中），它将对整个函数可见。但是，从2015年的ECMAScript，let并const声明允许您创建块范围变量。

运营商

JavaScript的数字运算符+，-，*，/和%其是余数运算符（这是不一样的模数。）值是使用分配的=，并且还存在化合物赋值语句如+=和-=。这些扩展到。x = x operator y

x += 5;
x = x + 5;

您可以分别使用++和--增加和减少。这些可以用作前缀或后缀运算符。

该+运算符还会执行字符串连接：

'hello' + ' world'; // "hello world"

如果将一个字符串添加到数字（或其他值），则首先将所有内容都转换为字符串。这可能会把你赶出去：

'3' + 4 + 5;  // "345"
 3 + 4 + '5'; // "75"

向某个东西添加一个空字符串是将其转换为字符串本身的一种有用方法。

比较在JavaScript中可以使用由<，>，<=和>=。这些都适用于字符串和数字。平等不那么简单。如果给它不同的类型，double-equals运算符会执行类型强制转换，有时会出现一些有趣的结果：

123 == '123'; // true
1 == true; // true

为了避免类型强制，使用triple-equals操作符：

123 === '123'; // false
1 === true;    // false

也有!=和!==运营商。

JavaScript也具有按位操作。如果你想使用它们，他们就在那里。

控制结构

JavaScript有一套类似于C系列其他语言的控制结构。条件语句由if和支持else; 如果你喜欢，你可以将它们链接在一起：

var name = 'kittens';
if (name == 'puppies') {
  name += ' woof';
} else if (name == 'kittens') {
  name += ' meow';
} else {
  name += '!';
}
name == 'kittens meow';

JavaScript有while循环和do-while循环。第一个对基本循环很有用; 第二个用于循环的地方，你希望确保循环的主体至少执行一次：

while (true) {
  // an infinite loop!
}

var input;
do {
  input = get_input();
} while (inputIsNotValid(input));

JavaScript的for循环与C和Java中的循环相同：它允许您在单行上为循环提供控制信息。

for (var i = 0; i < 5; i++) {
  // Will execute 5 times
}

JavaScript还包含另外两个重要的for循环：for...of

for (let value of array) {
  // do something with value
}

和for...in：

for (let property in object) {
  // do something with object property
}

的&&和||运营商使用短路逻辑，这意味着它们是否会执行他们的第二个操作数是依赖于第一。这对于在访问属性之前检查空对象很有用：

var name = o && o.getName();

或者为了缓存值（当错误值无效时）：

var name = cachedName || (cachedName = getName());

JavaScript有条件表达式的三元运算符：

var allowed = (age > 18) ? 'yes' : 'no';

该switch语句可用于基于数字或字符串的多个分支：

switch (action) {
  case 'draw':
    drawIt();
    break;
  case 'eat':
    eatIt();
    break;
  default:
    doNothing();
}

如果你不添加一个break语句，执行将“落空”到一个新的水平。这很少是你想要的 - 实际上，如果你确实想要帮助调试，那么它特别值得为评论标记故意传递一个评论：

switch (a) {
  case 1: // fallthrough
  case 2:
    eatIt();
    break;
  default:
    doNothing();
}

默认子句是可选的。如果您愿意，您可以在开关部分和案例中都有表达式; 使用===运营商的两者之间进行比较：

switch (1 + 3) {
  case 2 + 2:
    yay();
    break;
  default:
    neverhappens();
}

对象

JavaScript对象可以被认为是名称 - 值对的简单集合。因此，它们类似于：

  • Python中的字典。
  • 在Perl和Ruby中的哈希。
  • C和C ++中的散列表。
  • Java中的HashMaps。
  • PHP中的关联数组。

这个数据结构被广泛使用的事实证明了它的多功能性。由于JavaScript中的所有内容（bar核心类型）都是对象，所以任何JavaScript程序自然都会涉及大量的哈希表查找。他们太快了，这是一件好事！

“名称”部分是一个JavaScript字符串，而值可以是任何JavaScript值 - 包括更多对象。这使您可以构建任意复杂的数据结构。

有两种基本的方法来创建一个空对象：

var obj = new Object();

和：

var obj = {};

这些在语义上是等价的; 第二个被称为对象字面语法并且更方便。这种语法也是JSON格式的核心，应该始终是首选。

对象文字语法可以用来完整地初始化一个对象：

var obj = {
  name: 'Carrot',
  for: 'Max', // 'for' is a reserved word, use '_for' instead.
  details: {
    color: 'orange',
    size: 12
  }
};

属性访问可以链接在一起：

obj.details.color; // orange
obj['details']['size']; // 12

以下示例创建一个对象原型，Person 以及该原型的一个实例  you。

function Person(name, age) {
  this.name = name;
  this.age = age;
}

// Define an object
var you = new Person('You', 24); 
// We are creating a new person named "You" aged 24.

一旦创建，对象的属性可以通过以下两种方式之一再次访问：

// dot notation
obj.name = 'Simon';
var name = obj.name;

和...

// bracket notation
obj['name'] = 'Simon';
var name = obj['name'];
// can use a variable to define a key
var user = prompt('what is your key?')
obj[user] = prompt('what is its value?')

这些在语义上也是等同的。第二种方法的优点是，该属性的名称以字符串形式提供，这意味着它可以在运行时进行计算。但是，使用此方法可防止应用某些JavaScript引擎和缩小器优化。它也可以用来设置和获取名称为保留字的属性：

obj.for = 'Simon'; // Syntax error, because 'for' is a reserved word
obj['for'] = 'Simon'; // works fine

从ECMAScript 5开始，保留字可以用作对象属性名称“in the buff”。这意味着在定义对象文字时，它们不需要在引号中“穿衣”。请参阅ES5 规格。

有关对象和原型的更多信息，请参见Object.prototype。有关对象原型和对象原型链的解释，请参阅继承和原型链。

从ECMAScript 2015开始，对象键可以在创建时由变量使用括号表示来定义。{[phoneType]: 12345}是可能的，而不是仅仅var userPhone = {}; userPhone[phoneType] = 12345。

数组

JavaScript中的数组实际上是一种特殊类型的对象。它们的工作方式与常规对象非常相似（数值属性自然只能使用[]语法访问），但它们有一个名为' length'的魔术属性。这总是比数组中的最高索引多一个。

创建数组的一种方法如下：

var a = new Array();
a[0] = 'dog';
a[1] = 'cat';
a[2] = 'hen';
a.length; // 3

更方便的表示法是使用数组文字：

var a = ['dog', 'cat', 'hen'];
a.length; // 3

请注意，这array.length不一定是数组中的项目数。考虑以下：

var a = ['dog', 'cat', 'hen'];
a[100] = 'fox';
a.length; // 101

记住 - 数组的长度比最高索引多一个。

如果查询不存在的数组索引，则会得到一个值undefined ：

typeof a[90]; // undefined

如果你把上面的约[]和length考虑，你可以用下面的迭代数组for循环：

for (var i = 0; i < a.length; i++) {
  // Do something with a[i]
}

ECMAScript中引入了更简洁for...of用于迭代对象，如数组循环：

for (const currentValue of a) {
  // Do something with currentValue
}

你也可以遍历使用数组for...in循环，然而，这并不遍历数组元素，但数组索引。此外，如果有人添加了新的属性Array.prototype，它们也会被这样的循环迭代。因此这种循环类型不推荐用于数组。

迭代使用ECMAScript 5添加的数组的另一种方法是forEach()：

['dog', 'cat', 'hen'].forEach(function(currentValue, index, array) {
  // Do something with currentValue or array[index]
});

如果你想追加一个项目到一个数组，只需像这样做：

a.push(item);

数组带有许多方法。另请参阅数组方法的完整文档。

方法名称 描述
a.toString() toString()以逗号分隔每个元素返回一个字符串。
a.toLocaleString() toLocaleString()以逗号分隔每个元素返回一个字符串。
a.concat(item1[, item2[, ...[, itemN]]]) 返回一个新的数组，其中添加了项目。

功能

除了对象外，函数是理解JavaScript的核心组件。最基本的功能不能简单得多：

function add(x, y) {
  var total = x + y;
  return total;
}

这演示了一个基本功能。JavaScript函数可以使用0个或多个命名参数。函数体可以包含尽可能多的语句，并可以声明自己的变量，这些变量对于该函数来说是本地的。该return语句可以随时用于返回值，终止该函数。如果没有使用返回语句（或没有值的空返回），则JavaScript返回undefined。

命名的参数比其他任何东西都更像是指南。你可以调用一个函数而不需要传递它所期望的参数，在这种情况下，它们将被设置为undefined。

add(); // NaN
// You can't perform addition on undefined

你也可以传入比函数所期望的更多的参数：

add(2, 3, 4); // 5
// added the first two; 4 was ignored

这看起来可能有点愚蠢，但函数可以在其body内部调用一个额外的变量arguments，这是一个类似数组的对象，它持有所有传递给该函数的值。让我们重新编写add函数，根据需要获取尽可能多的值：

function add() {
  var sum = 0;
  for (var i = 0, j = arguments.length; i < j; i++) {
    sum += arguments[i];
  }
  return sum;
}

add(2, 3, 4, 5); // 14

这实际上没有比写作更有用2 + 3 + 4 + 5。我们来创建一个平均函数：

function avg() {
  var sum = 0;
  for (var i = 0, j = arguments.length; i < j; i++) {
    sum += arguments[i];
  }
  return sum / arguments.length;
}

avg(2, 3, 4, 5); // 3.5

这很有用，但看起来有点冗长。为了更多地减少这些代码，我们可以考虑通过Rest参数语法替换参数数组的使用。通过这种方式，我们可以将任意数量的参数传递给函数，同时保持我们的代码最小。在其余的操作参数在函数参数列表中使用的格式：...变量，它将变量中包含的未捕获的论点，即调用函数与整个列表。我们还将用for ... of循环替换for循环以返回变量中的值。

function avg(...args) {
  var sum = 0;
  for (let value of args) {
    sum += value;
  }
  return sum / args.length;
}

avg(2, 3, 4, 5); // 3.5
在上面的代码中，变量args保存了传递给函数的所有值。

值得注意的是，无论何处将其余参数运算符放置在函数声明中，它都将在声明后存储所有参数，但不会在之前存储。即函数 avg（firstValue，... args） 将存储firstValue变量中传递给函数的第一个值和args中剩余的参数。这是另一种有用的语言功能，但它确实会导致我们遇到新的问题。该avg()函数使用逗号分隔的参数列表 - 但如果您想查找数组的平均值，该怎么办？你可以重写这个函数，如下所示：
function avgArray(arr) {
  var sum = 0;
  for (var i = 0, j = arr.length; i < j; i++) {
    sum += arr[i];
  }
  return sum / arr.length;
}

avgArray([2, 3, 4, 5]); // 3.5

但是能够重用我们已经创建的函数会很好。幸运的是，JavaScript允许您使用apply()任意函数对象的方法调用具有任意参数数组的函数。

avg.apply(null, [2, 3, 4, 5]); // 3.5

第二个参数apply()是要用作参数的数组; 第一个将在稍后讨论。这强调了功能也是对象的事实。

在函数调用中使用spread运算符可以达到相同的结果。

例如： avg(...numbers)

JavaScript可以让你创建匿名函数。

var avg = function() {
  var sum = 0;
  for (var i = 0, j = arguments.length; i < j; i++) {
    sum += arguments[i];
  }
  return sum / arguments.length;
};

这在语义上等同于function avg()表单。它非常强大，因为它可以让你在任何你通常放置表达式的地方放置一个全功能定义。这使各种聪明的技巧。这里有一种“隐藏”一些局部变量的方法 - 比如C：

var a = 1;
var b = 2;

(function() {
  var b = 3;
  a += b;
})();

a; // 4
b; // 2

JavaScript允许您递归调用函数。这对处理树结构特别有用，例如浏览器DOM中的树结构。

function countChars(elm) {
  if (elm.nodeType == 3) { // TEXT_NODE
    return elm.nodeValue.length;
  }
  var count = 0;
  for (var i = 0, child; child = elm.childNodes[i]; i++) {
    count += countChars(child);
  }
  return count;
}

这突出显示了匿名函数的一个潜在问题：如果他们没有名字，你如何递归地调用它们？JavaScript可以让你为此命名函数表达式。您可以使用命名的IIFE（立即调用函数表达式），如下所示：

var charsInBody = (function counter(elm) {
  if (elm.nodeType == 3) { // TEXT_NODE
    return elm.nodeValue.length;
  }
  var count = 0;
  for (var i = 0, child; child = elm.childNodes[i]; i++) {
    count += counter(child);
  }
  return count;
})(document.body);

提供给上述函数表达式的名称仅适用于函数自己的作用域。这允许引擎完成更多优化，并产生更多可读代码。该名称也显示在调试器和一些堆栈跟踪中，这可以为您节省调试时间。

请注意，JavaScript函数本身就是对象 - 就像JavaScript中的其他所有内容一样 - 您可以在它们上添加或更改属性，就像我们之前在对象部分中看到的一样。

自定义对象

有关JavaScript中面向对象编程的更详细讨论，请参阅面向对象的JavaScript简介。

在经典的面向对象编程中，对象是对数据进行操作的数据和方法的集合。JavaScript是一种基于原型的语言，不包含类语句，就像您在C ++或Java中找到的那样（这对于习惯于具有类语句的语言的程序员有时会产生混淆）。相反，JavaScript使用函数作为类。让我们考虑一个带有名字和姓氏字段的人物。名称显示的方式有两种：“倒数第一”或“倒数第一”。使用前面讨论过的函数和对象，我们可以像这样显示数据：

function makePerson(first, last) {
  return {
    first: first,
    last: last
  };
}
function personFullName(person) {
  return person.first + ' ' + person.last;
}
function personFullNameReversed(person) {
  return person.last + ', ' + person.first;
}

s = makePerson('Simon', 'Willison');
personFullName(s); // "Simon Willison"
personFullNameReversed(s); // "Willison, Simon"

这有效，但它很丑陋。您最终会在您的全局名称空间中创建数十个函数。我们真正需要的是一种将函数附加到对象的方法。由于函数是对象，所以这很简单：

function makePerson(first, last) {
  return {
    first: first,
    last: last,
    fullName: function() {
      return this.first + ' ' + this.last;
    },
    fullNameReversed: function() {
      return this.last + ', ' + this.first;
    }
  };
}

s = makePerson('Simon', 'Willison');
s.fullName(); // "Simon Willison"
s.fullNameReversed(); // "Willison, Simon"

这里有一些我们从未见过的this关键词。在函数内部使用，this指的是当前对象。这实际上意味着什么是通过您调用该函数的方式来指定的。如果您在对象上使用点表示法或括号表示法调用它，则该对象将变为this。如果点符号未用于调用，则this引用全局对象。

请注意，这this是错误的常见原因。例如：

s = makePerson('Simon', 'Willison');
var fullName = s.fullName;
fullName(); // undefined undefined

当我们fullName()单独调用而不使用时s.fullName()，this就会被绑定到全局对象。由于没有全局变量被调用first或者last我们undefined为每个变量获取。

我们可以利用this关键字来改善我们的makePerson功能：

function Person(first, last) {
  this.first = first;
  this.last = last;
  this.fullName = function() {
    return this.first + ' ' + this.last;
  };
  this.fullNameReversed = function() {
    return this.last + ', ' + this.first;
  };
}
var s = new Person('Simon', 'Willison');

我们引入了另一个关键字：new。new与...密切相关this。它创建一个全新的空对象，然后调用指定的函数，并this设置为该新对象。注意，虽然指定的函数this不会返回值，但仅会修改this对象。这new就是将this对象返回给调用网站。被设计为被调用的new函数称为构造函数。通常的做法是将这些功能大写为提醒他们打电话给他们new。

改进后的功能与fullName()单独调用仍然有相同的缺陷。

我们的人物变得越来越好，但他们仍然有一些丑陋的边缘。每次我们创建一个人物对象时，我们都在其中创建了两个全新的函数对象 - 如果这个代码是共享的，会不会更好？

function personFullName() {
  return this.first + ' ' + this.last;
}
function personFullNameReversed() {
  return this.last + ', ' + this.first;
}
function Person(first, last) {
  this.first = first;
  this.last = last;
  this.fullName = personFullName;
  this.fullNameReversed = personFullNameReversed;
}

这更好：我们只创建一次方法函数，并在构造函数中为它们分配引用。我们能做得比这更好吗？答案是肯定的：

function Person(first, last) {
  this.first = first;
  this.last = last;
}
Person.prototype.fullName = function() {
  return this.first + ' ' + this.last;
};
Person.prototype.fullNameReversed = function() {
  return this.last + ', ' + this.first;
};

Person.prototype是所有实例共享的对象Person。它构成查找链（它有一个特殊名称，“原型链”）的一部分：任何时候当你尝试访问Person未设置的属性时，JavaScript都会检查Person.prototype该属性是否存在。因此，Person.prototype通过该this对象赋给该构造函数的所有实例的任何东西都可用。

这是一个非常强大的工具。JavaScript允许您随时在程序中修改某些原型，这意味着您可以在运行时为现有对象添加额外的方法：

s = new Person('Simon', 'Willison');
s.firstNameCaps(); // TypeError on line 1: s.firstNameCaps is not a function

Person.prototype.firstNameCaps = function() {
  return this.first.toUpperCase();
};
s.firstNameCaps(); // "SIMON"

有趣的是，您还可以将内容添加到内置JavaScript对象的原型中。让我们添加一个方法来String反向返回该字符串：

var s = 'Simon';
s.reversed(); // TypeError on line 1: s.reversed is not a function

String.prototype.reversed = function() {
  var r = '';
  for (var i = this.length - 1; i >= 0; i--) {
    r += this[i];
  }
  return r;
};

s.reversed(); // nomiS

我们的新方法甚至可以处理字符串文字！

'This can now be reversed'.reversed(); // desrever eb won nac sihT

如前所述，原型构成了链条的一部分。该链的根源是Object.prototype，其方法包括toString()- 当您尝试将对象表示为字符串时调用此方法。这对调试我们的Person对象很有用：

var s = new Person('Simon', 'Willison');
s.toString(); // [object Object]

Person.prototype.toString = function() {
  return '<Person: ' + this.fullName() + '>';
}

s.toString(); // "<Person: Simon Willison>"

还记得如何avg.apply()有一个空的第一个参数？我们现在可以重新访问。第一个参数apply()是应被视为“ this' 的对象。例如，以下是一个简单的实现new：

function trivialNew(constructor, ...args) {
  var o = {}; // Create an object
  constructor.apply(o, args);
  return o;
}

这不是一个确切的复制品，new因为它没有设置原型链（这很难说明）。这不是你经常使用的东西，但是了解它很有用。在这个片段中，...args（包括省略号）被称为“ 休息参数 ” - 顾名思义，它包含了其余的参数。

调用

var bill = trivialNew(Person, 'William', 'Orange');

因此几乎等同于

var bill = new Person('William', 'Orange');

apply()有一个名为姐妹函数call，它可以让你设置this一个扩展的参数列表，而不是一个数组。

function lastNameCaps() {
  return this.last.toUpperCase();
}
var s = new Person('Simon', 'Willison');
lastNameCaps.call(s);
// Is the same as:
s.lastNameCaps = lastNameCaps;
s.lastNameCaps(); // WILLISON

内部功能

其他函数内部允许使用JavaScript函数声明。我们之前已经看到过这个，具有更早的makePerson()功能。JavaScript中嵌套函数的一个重要细节是它们可以访问其父函数范围内的变量：

function parentFunc() {
  var a = 1;

  function nestedFunc() {
    var b = 4; // parentFunc can't use this
    return a + b; 
  }
  return nestedFunc(); // 5
}

这在编写更多可维护的代码时提供了很大的实用性。如果函数依赖于其他函数对代码的任何其他部分无用，则可以将这些实用函数嵌套在将从别处调用的函数中。这使得全局范围内的函数数量减少，这总是一件好事。

这也是对全球变数诱惑的一个很好的反击。在编写复杂的代码时，通常很容易使用全局变量在多个函数之间共享值 - 这导致难以维护的代码。嵌套函数可以在其父代中共享变量，因此，如果有意义，可以使用该机制将函数耦合在一起，而不会污染全局名称空间 - “本地全局变量”（如果您愿意的话）。这项技术应谨慎使用，但这是一种有用的能力。

关闭

这使我们成为JavaScript必须提供的最强大的抽象之一 - 但也是最容易混淆的。这是做什么的？

function makeAdder(a) {
  return function(b) {
    return a + b;
  };
}
var x = makeAdder(5);
var y = makeAdder(20);
x(6); // ?
y(7); // ?

该makeAdder()函数的名称应该可以让它消失：它创建了新的“加法器”函数，每个函数在用一个参数调用时将其添加到它所创建的参数中。

这里发生的事情与之前内部函数发生的事情大致相同：在另一个函数内部定义的函数可以访问外部函数的变量。这里唯一的区别是外部函数已经返回，因此常识似乎会指示其局部变量不再存在。但是，他们也仍然存在-否则，加法器功能将无法正常工作。更重要的是，makeAdder()局部变量有两个不同的“副本” - 其中一个a是5，另一个  a是20。因此，函数调用的结果如下所示：

x(6); // returns 11
y(7); // returns 27

这是实际发生的事情。只要JavaScript执行一个函数，就会创建一个“范围”对象来保存在该函数中创建的局部变量。它使用作为函数参数传入的任何变量进行初始化。这与所有全局变量和函数所处的全局对象相似，但有一些重要的区别：首先，每次函数开始执行时都会创建一个全新的作用域对象，其次，与全局对象不同如访问this并为浏览器window），这些范围的对象不能直接从你的JavaScript代码访问。例如，没有用于迭代当前作用域对象的属性的机制。

所以当makeAdder()被调用时，一个范围对象被创建一个属性：a，这是传递给该makeAdder()函数的参数。makeAdder()然后返回一个新创建的函数。通常情况下，JavaScript的垃圾回收器会清理为makeAdder()此时创建的作用域对象，但返回的函数将引用返回到该作用域对象。结果，范围对象将不会被垃圾收集，直到不再有对makeAdder()返回的函数对象的引用。

Scope对象构成一个称为作用域链的链，类似于JavaScript对象系统所使用的原型链。

甲闭合是一个函数，并在它被创建的范围对象的组合。关闭使您可以保存状态 - 因此，它们通常可以用来代替物体。你可以找到几个很好的封闭介绍。
